# **本论文主要研究各主流神经网络预测股票价格**
论文大体由三部分构成：基础神经网络对比、神经网络优化（减枝、降维等）、方法集成
## 数据准备
本文选取了具有从2015.01.01-2025.12.31完整交易数据的210支股票，收集了这些股票的日线基本行情以及一些技术指标，具体指标将会在附录列出。
## 一、基础神经网络对比
本章中选取五种常见的预测方法进行对比：ARIMA、LSTM、KAN、GNN、Transformer。为保证可比性，均采用相同的训练集、验证集和测试集，都暂定为60日预测下一日。
## 二、神经网络降维、减枝
在第一章中，神经网络只是简单地预测，是否可以通过减枝等方法优化神经网络，让其预测更准确。
## 三、神经网络集成
对以上所用到的各神经网络进行集成，以提高准确率。
